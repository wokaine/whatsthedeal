## What's the Deal?
A site to share your best (or worst) meal deals

The site is available [here](https://whatsthedeal.freddiebutterfield.com)

### About
This is a revival of a project I started in 2022. The aim was to make a very small social media site where people could share what meal deal combinations they had. Back then, the project was running on Flask and MongoDB which ended up causing a lot of problems. Now that I have infrastructure I decided to revive the project using Django and PostgreSQL instead which works nicely.

### Features
- Uploading a meal deal (you can do this either with an account or anonymously)
- Viewing other people's meal deals
- Liking or disliking them (account required)
